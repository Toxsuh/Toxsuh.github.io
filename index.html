What's here?

This website is created with just HTML and CSS, and was created with extreme urgency for the need to put SOME viable work for my WDCC projects interview. That being said, finally getting a personal site up and running was long overdue. It's clearly still a work in progress, but I'd like to eventually build this out into the main focal point for everything I do in the tech industry.
